## 铜矿

给铜矿增加更多用途

- 工具：铜斧，铜镐，铜铲，铜锄
- 装备：铜头盔，铜胸甲，铜护腿，铜靴子

## 狱石矿

下界增加狱石矿，狱石矿可做

- 工具：狱石斧，狱石镐，狱石铲，狱石锄
- 装备：狱石头盔，狱石胸甲，狱石护腿，狱石靴子

生成于20-60高度范围内

## 紫水晶

增加紫水晶剑，伤害高，但耐久低，制作时自带锋利附魔
